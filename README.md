# nhentai Favorites Downloader

一個簡潔強大的 Chrome 擴充功能，讓您能夠批量下載 nhentai 收藏夾中的漫畫並**自動打包為 ZIP**。

---

## ✨ 功能特色

- 🎯 **批量下載**：在收藏夾頁面勾選多本漫畫，一鍵下載
- 📦 **ZIP 打包**：所有圖片自動打包為單一 ZIP 檔案
- 🎨 **美觀介面**：現代化的 UI 設計，與 nhentai 完美融合
- ⚡ **快速穩定**：直接在瀏覽器中運行，無需額外設定

---

## 📥 安裝方法

### Chrome / Edge

1. 下載本擴充功能資料夾到您的電腦
2. 在 Chrome 網址列輸入 `chrome://extensions/`
3. 開啟右上角的「**開發人員模式**」
4. 點擊左上角的「**載入未封裝項目**」
5. 選擇 `nhentai_extension` 資料夾
6. 完成！您會在工具列看到擴充功能圖示

---

## 🚀 使用教學

### 批量下載收藏夾

1. 登入您的 nhentai 帳號
2. 前往 **收藏夾頁面** (Favorites)
3. 您會看到每本漫畫左上角出現 **勾選框**
4. 勾選想要下載的漫畫（或點擊「全選」）
5. 點擊右下角的 **「📥 下載已選」** 按鈕
6. 等待下載完成！

### 下載單本漫畫

1. 進入任意漫畫頁面 (`/g/xxxxx`)
2. 在資訊區域會出現 **「下載此漫畫」** 按鈕
3. 點擊即可開始下載

## 📁 下載位置

所有漫畫會下載到您的 **預設下載資料夾** 中的 `nhentai` 子目錄：

```
Downloads/
└── nhentai/
    ├── [漫畫標題1]/
    │   ├── 001.jpg
    │   ├── 002.jpg
    │   └── ...
    └── [漫畫標題2]/
        └── ...
```
